# 🚀 DEPLOY AUTOMÁTICO NO VERCEL

## Para fazer o deploy automático, preciso do seu Vercel Token.

### Como obter o token:

1. **Acesse:** https://vercel.com/account/tokens
2. **Clique em:** "Create Token"
3. **Nome:** digite "deploy-quebrando-ciclo"
4. **Scope:** deixe em "Full Account"
5. **Expiration:** escolha "No Expiration"
6. **Clique em:** "Create"
7. **Copie o token** que aparece na tela

### Depois, forneça o token aqui e farei o deploy automático!
